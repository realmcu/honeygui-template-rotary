# RotaryTemplate

Created from template: **Rotary Screen**

Circular UI design optimized for rotary knob interaction.

## Project Info
- APP ID: com.honeygui.rotary
- Resolution: 480X480
- Target Engine: HoneyGUI
- Min SDK: API 2: HoneyGUI V1.1.0
- Pixel Mode: ARGB8888

## Features
- Circular layout with center time display
- Four-direction quick access icons (Phone, Messages, Music, Settings)
- Battery indicator
- Step counter
- Optimized for rotary knob navigation

## Getting Started

1. Open the HML file in `ui/main/RotaryMain.hml`
2. Design your UI in the visual designer
3. Generate code and compile
